Main

Main is the container that has all the main content of the website, which differs evry page. Main contains following items

  • p
  • a
  • ul,ol,li
  • img

p tag is used to write paragraphs or sentences in html.

a is anchor tag and it is used to make links withing html, a href= "" (em tag was used to do this) links pages within the the html and also used links to other webpages.

List are used to contain links, listing things etc. There are different type of list, most used ones are unordered list ul(used to list items without a set order), ordered list ol(used for contet that ahs a n order), and within ul and ol sits li

img is to display imags on the website. img src="images/about-me.jpg"alt=""

figcaption is used the to give an images a caption to provide information.

img and figcaption together make a figure.

Blockqoteand cite are used when there is a quote in the content. example
